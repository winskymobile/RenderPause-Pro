偏好设置 · 三种不同结构

不是同一套三 Tab 换皮，而是三种信息架构。点开对比后告诉我选哪一个（或混搭）。

推荐 · 名单优先

A · 左右分栏

左名单、右详情/全局/日志。日常改名单最快，设置不抢戏。

 最像系统设置

B · 单页纵向

无 Tab。分组滚动：状态 → 通用 → 权限 → 名单 → 活动。

 数据表感

C · 工具栏 + 大表

顶栏状态 + 大名单表格，底栏全局摘要。名单多时最清楚。

 旧稿对照

D · 三 Tab（当前方向）

应用名单 / 设置 / 最近活动。你已反馈不好看，作对照用。

功能不变：opt-in 名单、全局后台秒数、隐藏/最小化、辅助功能、今日优化次数。仅布局与层级不同。
